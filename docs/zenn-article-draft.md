## Future Self Interview とは

**自撮り1枚をアップロードすると、AI が推定した未来の年齢の自分とチャットできる Web アプリ** です。

左に年齢の時系列。右に対話。肌の測定データを根拠に、未来の自分が今の自分へ語りかけます。

| | 一般的な老化シミュレーション | Future Self Interview |
| --- | --- | --- |
| 体験の終わり方 | 老けた顔を見て終わり | **老けた自分と話し続けられる** |
| 感情の設計 | 「今のうちにケアを」という警告 | 未来からの振り返りと応援 |
| 肌データの使い方 | 商品購入への誘導 | 対話の根拠になる具体的アドバイス |

コンセプトは一言でこうです。

> **老いを見せて終わりにしない。未来の自分と話すところまで持っていく。**

## なぜ作ったか

老化シミュレーションアプリは、だいたいこういう体験です。

1. 自撮りをアップする
2. シワの増えた未来の顔が出る
3. 「今のうちにスキンケアを」というメッセージで終わる

技術としてはすごい。でも **未来の自分は「警告」としてしか登場しない**。

今の選択を、あとから後悔しないか——肌のことも、キャリアのことも、そういう問いだ。老けた顔を **見るだけ** では消化しきれない。

だから Future Self Interview では、老化の先に **会話** を置きました。

- 何歳になった自分なのか、写真で見える
- その年齢の自分が、今の自分に直接話しかけてくる
- 肌の数値を根拠に、「あのときこうしておけばよかった」が具体的に返ってくる

占いでも、汎用チャットでもない。**自分の顔を起点にした、未来の自分へのインタビュー** です。

---

## 使い方

ブラウザだけで動きます。アカウント登録は不要です。

```
① API キーを入力（YouCam + Claude）
   ↓
② 自撮りをアップロード（男性 / 女性を選択）
   ↓
③ 「未来の自分を呼び出す」を押す（30〜60秒）
   ↓
④ 年齢の時系列と肌データが表示される
   ↓
⑤ 未来の自分（最も老けた年齢）とチャット開始
```

API キーはブラウザ内だけで処理します。サーバーには送信しません。

## 体験の3つの柱

### 1. あなたの時系列——12歳から70歳まで

左パネルに、現在の推定年齢と、AI が生成した過去・未来の顔が並びます。

- **現在**（例: 31歳）……アップロードした自撮り
- **12歳、20代、40代……**……Aging Generator が生成した年齢別の顔
- **最年長**（例: 70歳）……対話相手となる「未来の自分」

すべて YouCam Aging Generator API が返した画像をそのまま表示しています。1枚の自撮りから、人生の顔の変化が一覧できます。

![年齢の時系列 — スクショ挿入予定](images/aging-timeline.png)

### 2. 肌データ——未来の自分が語る根拠

同じ左パネルに、Skin Analysis API の測定値が4項目で表示されます。

- 水分量
- シワ
- 毛穴
- ニキビ

100点満点で、低いほど気になる状態。未来の自分はこの数値を見て、具体的なアドバイスを混ぜながら話します。

「保湿をサボったな」「紫外線対策、もっと早く始めればよかった」——恐怖で脅すのではなく、**未来から振り返る口調** で返ってきます。

![肌データの表示 — スクショ挿入予定](images/skin-data.png)

### 3. 対話——未来の自分が話しかけてくる

右パネルがチャット UI です。ヘッダーには「未来の自分（70歳）と話しています」と表示されます。

最初のメッセージから、未来の自分が先に話しかけてきます。そのあとは自由に質問できます。

プリセットも用意しています。

- 「転職しようか迷ってるんだけど、どうだった？」
- 「もっとやっておけばよかったことって何？」
- 「今の自分に一言ちょうだい」

![未来の自分との対話 — スクショ挿入予定](images/interview-chat.png)

## プロダクトとしてのこだわり

### 見るだけで終わらせない

老化シミュレーションの多くは、生成された顔を見せて体験が完結します。

Future Self Interview は、その先に **対話** があります。老いを「確認」するのではなく、**未来の自分と向き合う** ための設計です。

### 肌データは「測定値」で渡す

肌分析 API は、測定値（`raw_score`）と画面表示用の点数（`ui_score`）の2種類を返します。

Future Self Interview では、未来の自分に **測定値だけ** を渡しています。ユーザー向けに調整された点数ではなく、API が実際に計測した数値を根拠に話してもらうためです。

「こういう数値になっていた」という事実ベースの振り返りに寄せたいからです。

### 性別に合った口調

アップロード時に男性 / 女性を選べます。

未来の自分の一人称や口調が変わります。男性なら「俺」で率直に。女性なら「私」で温かく。同じ肌データでも、話し方が自分に馴染むようになっています。

### プライバシー

YouCam API キーと Claude API キーは **sessionStorage** にだけ保存されます。タブを閉じれば消えます。サーバー側の処理はありません。

## こんなときに使える

### 人生の選択に迷っているとき

転職、引っ越し、続けるかやめるか。正解は返ってきません。でも **70歳の自分に聞いてみる** という行為自体が、考えを整理するきっかけになります。

### スキンケアを続ける理由がほしいとき

「今やらないとヤバい」という恐怖ではなく、「未来の自分が後悔していた」という **振り返り** として肌の話が返ってきます。脅しではなく、習慣づくりの動機になります。

### 自分と向き合う時間がほしいとき

毎日使うアプリではありません。人生の節目に1回、自撮りをアップして未来の自分と話す——そういう **内省の入り口** として使えます。

### 自分の「年表」を眺めたいとき

12歳から70歳までの顔が並ぶ左パネルは、それだけでも見応えがあります。AI が推定した、あなたの顔の時間軸です。

---

## 実際に話してみて

「こんにちは。未来の自分に会えて嬉しい。」

そう送ると、70歳の自分が返事をくれます。

転職の話。後悔の話。今の肌の状態について、あの頃もっとこうしておけばよかった、という話。

写真は AI が生成した未来の顔です。でも **「未来の自分」というフレーム** が効く。

老けた顔を見てドキッとする体験と、老けた自分と話す体験は、まったく違います。後者は、どこか落ち着く。

## 技術構成（概要）

実装の詳細は別途書く予定ですが、構成だけ共有します。

| レイヤー | 技術 |
| --- | --- |
| フロント | Next.js + TypeScript + Tailwind CSS |
| 老化画像 | YouCam Aging Generator API v2.0 |
| 肌分析 | YouCam Skin Analysis API v2.1（HD 5項目） |
| 対話 | Claude API（ブラウザから直接呼び出し） |
| デプロイ | 静的ホスティング（サーバーサイド処理なし） |

自撮り1枚に対して、老化生成と肌分析を **並行実行** し、完了後に Claude が未来の自分として初回メッセージを生成します。
